Dotfiles
========

To overwrite `~/.emacs.d`, `~/.tmux.conf`, `~/.vim`, and `~/.vimrc`:

    rake activate

To build vim with command-t:

    rbenv shell <ruby-version-you-like>
    brew install vim
    rake compile_command_t
